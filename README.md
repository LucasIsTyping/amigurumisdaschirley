# 🧶 Vitrine de Amigurumis

Este projeto consiste em uma landing page desenvolvida para apresentar amigurumis artesanais de forma simples, visual e acessível na internet.

Além da parte visual, o projeto conta com um script em Python que automatiza o cadastro dos produtos, facilitando a atualização do conteúdo sem a necessidade de editar manualmente o código da página.

## 🚀 Funcionalidades

* Página web (landing page) para exibição dos amigurumis
* Layout simples, focado na apresentação visual dos produtos
* Leitura dinâmica dos dados via arquivo JSON
* Script em Python para adicionar novos produtos de forma prática
* Estrutura leve e fácil de manter

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JSON
* Python

## 💡 Como funciona

A landing page consome um arquivo JSON que contém os dados dos amigurumis (como nome, descrição e imagem).

Para facilitar a manutenção, foi desenvolvido um script em Python que permite adicionar novos itens ao JSON de maneira automatizada, evitando erros e agilizando o processo de atualização.

## 📦 Como usar

### 1. Adicionar novos amigurumis

Execute o script Python para cadastrar um novo produto

Preencha as informações solicitadas e o item será automaticamente incluído no arquivo JSON.

### 2. Visualizar a vitrine

Basta abrir o arquivo `index.html` no navegador para visualizar a página com os produtos atualizados.
